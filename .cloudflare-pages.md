# Cloudflare Pages — configuração da Tríade 56

## Build

- **Framework preset**: None (deixa em branco)
- **Build command**:
  ```
  npm install && npm run build --workspace=@site-forge/shared && npm run build --workspace=@site-forge/web
  ```
- **Build output directory**: `apps/web/dist`
- **Root directory**: (em branco)

## Environment variables (Production)

- `NODE_VERSION` = `22`
- `VITE_TRIADE_API` = `https://misscanvas.com/api/insta-site` (ou o endpoint do seu worker)
- `VITE_WHATSAPP` = SEU número com DDI, só dígitos (ex.: `5511987654321`)

## Custom domain

- `triade56.com` → **Custom domains** → **Set up a custom domain**
- Opcionalmente `www.triade56.com` também

## Testar

Depois do primeiro deploy:
1. Abre a URL provisória (algo tipo `triade56.pages.dev`) pra ver se tudo carregou.
2. Cola um @ e clica INICIAR IMERSÃO — deve chamar o worker MissCanvas e devolver um link de preview.
3. Se der erro de CORS: garanta que `handleInstaSite` no worker retorna `Access-Control-Allow-Origin: *` (ou `https://triade56.com`).

# Configuração do Cloudflare Pages

Quando você conectar este repo no Cloudflare Pages, use estas configurações:

- **Framework preset**: None (deixe em branco)
- **Build command**: `npm install && npm run build --workspace=@site-forge/shared && npm run build --workspace=@site-forge/web`
- **Build output directory**: `apps/web/dist`
- **Root directory**: (deixe em branco — raiz do repo)
- **Node version**: 22 (adicione uma variable `NODE_VERSION` = `22` na aba Environment variables)

Depois que der o primeiro deploy, o Cloudflare vai te dar uma URL tipo `aurahype.pages.dev`. Pra ligar no seu domínio `aurahype.com`, vá em **Custom domains** → **Set up a custom domain** → digite `aurahype.com`. O Cloudflare cuida do DNS automaticamente já que o domínio está na mesma conta.
